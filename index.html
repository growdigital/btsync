  4. Set up hostname

  6. Enable DigitalOcean backups? $2 per month

2. Admin files

  1. Organise your files and folders on one computer. It is important to have a simple yet flexible file structure.
  2. Install Sync.
  3. Add folders to Sync.
  4. Copy link. For security, the BitTorrent Sync server should have READ ONLY permission.
  5. On the BitTorrent Sync server browser interface, click Manual Connection and paste in link.
  6. Connect to the folder. By default, all btsync folders are stored in /shared/.
  7. Approve the connection. That’s it :)

3. Large media files

When you set up your Mac Mini as a media server, follow the same process by installing Sync, adding the Media folder to Sync, then sharing it READ ONLY with BitTorrent Sync server. This way, the BitTorrent Sync server acts as a “mirror” of the Mac Mini Media library. That is, the BitTorrent Sync server will have all the hi-res images on it.

The BitTorrent Sync server has about 25GB of hard disk space, so there should be enough there for a while. It can be increased later by resizing the droplet in DigitalOcean.

Now when you share the Media folder from Sync on the Mac Mini with your laptops, make sure to turn on Selective Sync. That’s the whole point of having a Pro license!

Sync on a mobile has Selective Sync turned on by default, but on a desktop app, it’s Pro only.

4. Setup subsequent devices

As fldcrft.uk is acting as a dumb, read-only mirror of all shared folders, you need to share folders from your individual computers.

It’s best to use Advanced folders, as you have more control over peer management, ie what you share with whom. Always err on Read Only permissions, for obvious reasons!
